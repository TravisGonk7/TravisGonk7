What Is Bootstrap?

Bootsrap Is A Front-end framework that developes mobile first websites with a prebuilt responsive grid that groupes properties into base-classes which are named Components.

How does Bootstrap work?

Bootstrap uses a 12 column responsive grid system that includes breakpoints to manage your layout simillar to CSS. The use of base-classes is what you use to build your websites.
